# teal
Visualize trace files.

Installing
----------

`git clone https://github.com/gear-genomics/teal.git`

`cd teal`

`make all`

Install Web Interface
---------------------

`sudo apt install python python-pip`

`pip install flask flask_cors`

Running local
-------------

`python server/server.py`

Open in Browser:
`client/index.html`
